# Grow Book Log

오늘의 독서를 한 장의 카드로 만들어주는 웹 도구.

책 제목, 페이지, 시간을 입력하면 Strava 공유 카드처럼 깔끔한 PNG 카드를 만들어줍니다.

## 사용법

1. `index.html`을 브라우저로 열기
2. 책 정보와 오늘 읽은 페이지/시간 입력
3. "PNG 다운로드" 클릭

## 특징

- 단일 HTML 파일 — 의존성 없음
- 클라이언트 사이드 렌더링 — 입력 데이터가 외부로 안 나감
- 가로형 / 세로형 두 가지 레이아웃
- 투명 / 검정 두 가지 배경 모드
- 사각나선으로 책 진행도 시각화
